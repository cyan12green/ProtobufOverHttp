# ProtobufOverHttp

Sample code that shows how to send a protobuf message over http.


Requirements:
----------------
- [libhttpserver](https://github.com/etr/libhttpserver/tree/master)
- [cURLpp](https://github.com/jpbarrette/curlpp)

Building:
----------

```
mkdir -p cmake/build  
cmake -B cmake/build
cd cmake/build
make
```
